<h2>📋 Descripción</h2>

<p>
  <strong>PokeCircuit</strong> es una plataforma creada para jugadores competitivos de Pokémon que desean organizar sus equipos, registrar resultados de torneos y obtener estadísticas detalladas de su desempeño.
</p>

<p>
  La aplicación reúne en un solo lugar herramientas para la gestión de equipos, generación de teamsheets, seguimiento de torneos y análisis de rendimiento, ayudando a jugadores de todos los niveles a prepararse para la competición.
</p>

<h2>✨ Características</h2>

<ul>
  <li>🎯 Creación y administración de equipos competitivos.</li>
  <li>📄 Generación automática de Teamsheets listas para torneos.</li>
  <li>🏆 Registro de participaciones en torneos mediante RUN de jugador.</li>
  <li>📊 Estadísticas personales y análisis de Win Rate.</li>
  <li>📈 Seguimiento histórico de resultados y rendimiento.</li>
  <li>🔍 Exploración de equipos utilizados por la comunidad.</li>
  <li>🌎 Soporte para múltiples idiomas.</li>
  <li>⚡ Actualizaciones constantes según las regulaciones y cambios del metajuego.</li>
</ul>

<h2>🎯 Objetivo</h2>

<p>
  PokeCircuit busca convertirse en la plataforma de referencia para la comunidad competitiva de Pokémon, proporcionando herramientas que faciliten la preparación para torneos, el análisis de estrategias y el seguimiento del progreso de cada entrenador.
</p>
